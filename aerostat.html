Aerostat App for Android OS

Мобильное приложение Android для подкаста "Аэростат" Радио Аквариум

Download this project as a .zip file Download this project as a tar.gz file

Добро пожаловать на веб-сайт посвящённый мобильному приложению "Аэростат" Радио Аквариум для Android OS

Приложение в настоящее время находится на самой начальной стадии разработки и поэтому пока недоступно для скачивания. Следите за новостями!

Новости

20.06.2014 Версия 0.3 (Альфа)

  • Обеспеченна минимальная базовая функциональность - приложение выводит на экран список Аэростатов с названиями и датами выхода в эфир. По нажатию, выбранный выпуск загружается в медиаплейер и автоматически проигрывается. Приложению требуется доступ к Интернет, так как проигрываемые подкасты не хранятся локально на устройстве, а загружаются в реальном времени с сервера http://aerostat.rpod.ru/.
